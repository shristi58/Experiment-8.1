# Practice 1 - Authentication System (React Login Form)

##  Objective
Build a simple **Login Form** in React using the **useState** hook to manage form data and basic validation.

---

##  Features
- Two input fields: **Username** and **Password**
- Validation for empty fields
- Displays an error message when inputs are missing
- Logs entered credentials in the console on successful submission
- Simple, user-friendly UI

---

## How to Run
1. Open the project folder in VS Code or Nimbus.
2. Run the following commands:

```bash
npm install
npm start
